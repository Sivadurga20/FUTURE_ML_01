# 📊 Sales Forecasting Project

## 🔹 Objective
This project predicts future sales using historical data.

## 🔹 Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 🔹 Steps
1. Data Loading
2. Data Visualization
3. Feature Engineering
4. Model Training (Linear Regression)
5. Forecasting Future Sales

## 🔹 Output
- Sales trend visualization
- Future 7 days prediction
- Model evaluation (MAE)

## 🔹 Conclusion
The model successfully predicts future sales trends based on past data.
